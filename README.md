# Flutter + PHP E-Commerce Project

## Description
Mobile application built with Flutter and PHP backend.

## Technologies
- Flutter
- PHP
- MySQL
- Firebase (Notifications)
- Google Maps

## Project Structure
- frontend_flutter/ : Flutter app
- backend_php/ : PHP backend

## Setup
### Backend
- Run backend on XAMPP
- Import database
- Add Firebase service account JSON

### Frontend
- Run flutter pub get
- Add Firebase configuration files
- Add Google Maps API key

## Notes
Sensitive keys and configuration files were removed for security reasons.
